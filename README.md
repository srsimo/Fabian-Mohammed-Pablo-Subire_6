# Fabian-Mohammed-Pablo-Subire_6
Proyecto ERP - Módulo de Control de Clientes, Productos y Pedidos para Supermercado. Grupo 6. Interfaz con Tkinter, base de datos SQLite3, análisis gráfico y exportación a Excel.

## Descripción del Proyecto

Módulo ERP para la gestión de un supermercado. Permite:
- Gestionar clientes, productos y pedidos
- Conexión a base de datos SQLite3
- Interfaz gráfica con Tkinter
- Análisis gráfico de datos
- Exportación de resultados a Excel

## Estructura del Proyecto

```
├── ui/                 # Interfaz gráfica con Tkinter
├── db/                 # Módulo de base de datos SQLite3
├── graphics/           # Módulos de análisis gráfico
├── utils/              # Funciones auxiliares
├── main.py             # Archivo principal de ejecución
└── README.md           # Este archivo
```
